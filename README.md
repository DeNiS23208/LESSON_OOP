# Домашнее задание: Введение в ООП 14.1

В данном домашнем задании была реализована система для работы с товарами и категориями. Задача заключалась в создании двух классов (`Product` и `Category`), а также в реализации функциональности для добавления товаров в категории и подсчета общего количества категорий и товаров.

## Что было сделано:

1. **Классы:**
   - **Product** — класс для представления товара с аттрибутами:
     - `name`: Название товара
     - `description`: Описание товара
     - `price`: Цена товара
     - `quantity`: Количество товара
   - **Category** — класс для представления категории товаров с аттрибутами:
     - `name`: Название категории
     - `description`: Описание категории
     - `products`: Список товаров в категории
     - Автоматически отслеживает количество категорий и товаров во всей системе.

2. **Основная функциональность:**
   - При добавлении нового объекта категории увеличивается общее количество категорий.
   - При добавлении товара в категорию увеличивается общее количество товаров во всей системе.
   - В классе `Category` добавлен метод `add_product`, который добавляет товар в категорию и обновляет общее количество товаров.

3. **Тестирование:**
   - Написаны тесты с использованием библиотеки `pytest`, которые проверяют:
     - Корректность инициализации объектов классов `Product` и `Category`.
     - Подсчет количества категорий и товаров в системе.
     - Добавление товаров в категории и обновление статистики.

4. **Покрытие кода:**
   - Использован плагин `pytest-cov` для проверки покрытия кода.
   - Реализован отчет о покрытии в формате HTML.

5. **Линтинг:**
   - Для проверки качества кода используется инструмент `flake8`, который помогает соблюдать стиль кода и избегать ошибок.
